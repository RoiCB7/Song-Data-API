# Song Data API
## Overview
This project contains code for API route querying an existing song database. Results are returned in JSON format

## Built With
- Javascript
- Node.js - JS Runtime
- Express - Routing
- SQLite 3
- Hosted and deployed on Render - https://song-data-api.onrender.com/
